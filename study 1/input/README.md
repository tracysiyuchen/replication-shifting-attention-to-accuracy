# Data File Required

This folder expects `study_1_data.csv` to run the replication notebook.

The file is not included in this repository due to participant privacy considerations.
To replicate, obtain the dataset from the original authors and place it here as:

```
study 1/input/study_1_data.csv
```

**Source:** Pennycook, G., et al. (2021). "Shifting attention to accuracy can reduce misinformation online." *Nature*, 592, 590-595.
