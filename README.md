# Replication of “Shifting Attention to Accuracy Can Reduce Misinformation Online”

SDS 530 Social Algorithms Final Project
Hua Deng, Grace Yin, Hayley Shin, Tracy Chen, Gerald Velasquez

## Overview

This repository contains replication project for the paper **“Shifting attention to accuracy can reduce misinformation online”** by Pennycook, Epstein, Mosleh, Arechar, Eckles, and Rand, published in *Nature* in 2021.

In this project, I aim to reproduce selected results from the paper using Python, focusing on the main experimental findings and their statistical patterns.

## Paper Information

- **Title:** Shifting attention to accuracy can reduce misinformation online
- **Authors:** Gordon Pennycook, Ziv Epstein, Mohsen Mosleh, Antonio A. Arechar, Dean Eckles, David G. Rand
- **Venue:** Nature, 2021
- **DOI:** 10.1038/s41586-021-03344-2
